My name is Cheng-Chia Huang. I am a graduate research assistant in Department of Geography at San Diego State University. In the past few years, I was working on multiple projects as a GIS analyst and programmer.

My current works focus on human dynamics and analysis of geographic information in social media. My expertise lies in GIS, analysis of big geo-social media, integrating spatial analysis with data mining and machine learning algorithms, and geography education.

GIS Projects


GIS & Health

This project aims at developing a dashboard which is able to monitor the mental health of residents with sentiment analysis.
This project won the Most Thorough Design and Development Award at 2017 San Diego Big Data Hackathon.

GIS & Gentrification (1)

This project takes advantage of Graphlab Create and NLTK, a python library, to group the Instagram posts in Salt Lake County by the text similarity.
This project tries to test if gentrifying areas have similar Instagram posts

GIS & Gentrification (2)

This project takes advantage Tensor Flow, a machine learning python library, to classify the Instagram photos in the core area of Sugar House,SLC.

GIS & Gentrification (3)

This project develops a tool written in PostgreSQL and Python. This tool is able to generate the gentrification typologies based on different existing methods and visualize them as maps.
